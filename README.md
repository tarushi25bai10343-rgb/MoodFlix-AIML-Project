# 🎬 MoodFlix AI Project

## 📌 Description
MoodFlix is an AI-based recommendation system built using Python and Prolog that suggests movies, songs, and aesthetics based on the user’s mood. It uses Prolog for logic and Python for use interaction, allowing users to choose a genre and get personalised suggestions. 

## 🛠️ Technologies Used
- Python
- Prolog
- PySwip

## ▶️ How to Run the Project

1. Install required library:
   pip install pyswip

2. Run the program:
   python main.py

3. Enter your mood and choose a genre.

## ⚠️ Requirements
- Python installed
- SWI-Prolog installed
