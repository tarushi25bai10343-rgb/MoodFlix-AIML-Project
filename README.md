# 🎬 MoodFlix AI Project

## 📌 Description
MoodFlix is an AI-based recommendation system built using Python and Prolog that suggests movies, songs, and aesthetics based on the user’s mood. It uses Prolog for logic and Python for use interaction, allowing users to choose a genre and get personalised suggestions. 

## 🛠️ Technologies Used
- Python - for app logic and user interface
- Prolog - for knowledge representation and reasoning
- PySwip - Python-Prolog integration
- Flask - web framework for the interactive interface

## ▶️ How to Run the Project

1. Install required library:
   pip install flask pyswip

2. Make sure SWI-Prolog is installed.

3. Run the program:
   python app.py

4. Open your browser and go to: http://127.0.0.1:5000/

5. Workflow:
   Select your mood (happy, sad, bored, stressed)
   Select a genre from the suggested options
   View your recommended movie, poster, song, and aesthetic 

## ⚠️ Requirements
- Python installed
- SWI-Prolog installed
- All posters must be in static/posters/
- Flask and PySwip libraries installed
