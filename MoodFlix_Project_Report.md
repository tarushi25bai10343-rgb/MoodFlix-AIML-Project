# 🎬 MoodFlix AI Project Report

## 📌 Introduction
MoodFlix is an AI-based recommendation system developed using Python and Prolog. It is designed to help users choose entertainment content based on their current mood. The system provides movie suggestions, song recommendations, and visual aesthetics to enhance the overall experience.

---

## ❓ Problem Statement
In daily life, people often feel confused about what to watch or listen to, especially when they are in a specific mood such as happy, sad, bored, or stressed. This decision-making process can take time and reduce enjoyment.

---

## 💡 Proposed Solution
To solve this problem, MoodFlix suggests suitable content based on the user’s mood. It maps moods to different genres and then recommends:
- Movies 🎬  
- Songs 🎵  
- Aesthetic themes 🎨  

The system also allows users to choose a genre manually, making it more interactive and personalized.

---

## ⚙️ Technologies Used
- Python (for user interaction and program execution)  
- Prolog (for logic-based knowledge representation)  
- PySwip (to connect Python with Prolog)  

---

## 🔄 Working of the System
1. The user enters their mood (happy, sad, bored, or stressed).  
2. The system retrieves relevant genres using Prolog rules.  
3. The user selects a preferred genre from the list.  
4. The system randomly suggests:
   - A movie  
   - A song  
   - An aesthetic description  
5. The results are displayed to the user in a clean format.

---

## ⚠️ Challenges Faced
- Understanding and writing Prolog logic rules  
- Integrating Python with Prolog using PySwip  
- Handling multiple outputs and random selection  
- Avoiding errors due to syntax issues in Prolog
